# Product Core Rules

- Le produit est un annuaire éditorial augmenté par la data.
- Une source externe peut découvrir, enrichir, scorer ou contredire.
- Une source externe ne publie jamais seule.
- Une fiche publique doit toujours afficher :
  - statut chiens
  - conditions éventuelles
  - source principale
  - date de vérification
  - niveau de confiance
- Si l'information n'est pas explicite, utiliser `unknown`.
- Priorité constante : fiabilité > volume > sophistication.
- Le MVP doit rester frugal, maintenable et administrable.
- Refuser toute dérive vers réseau social, marketplace, app native ou fonctionnalités gadgets.
