# Data Ingestion Rules

- Toute donnée importée arrive comme candidat, brouillon ou source rattachée.
- Jamais de publication directe après import.
- Toujours conserver la source brute ou un extrait utile.
- Normaliser nom, catégorie, localisation et URLs avant comparaison.
- Dédupliquer avant création d'une nouvelle fiche.
- Quand une source est ambiguë, créer un besoin de review, pas une certitude.
- Toute extraction IA doit rester strictement factuelle.
- Si aucune mention explicite de la politique chien n'existe, le statut candidat = `unknown`.
