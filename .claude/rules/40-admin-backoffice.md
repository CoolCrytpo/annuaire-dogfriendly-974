# Admin Backoffice Rules

- L'admin est un outil métier, pas un décor.
- Prioriser vitesse, densité utile et clarté.
- Les workflows clés doivent être traitables en peu de clics :
  - créer / éditer fiche
  - valider contribution
  - fusionner doublons
  - gérer sources
  - revalider
- Ajouter des vues de comparaison quand une décision éditoriale est nécessaire.
- Prévoir des bulk actions seulement si elles réduisent vraiment la friction.
- Toute action critique doit laisser une trace dans l'audit log.
