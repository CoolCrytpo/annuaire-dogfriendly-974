# Frontend UX Rules

- Mobile-first obligatoire.
- Sobriété visuelle.
- Lisibilité maximale des informations critiques.
- Le statut chiens doit être visible immédiatement.
- La date de vérification et le niveau de confiance doivent être visibles sans scroller loin.
- Les filtres doivent rester courts et utiles.
- Pas de surcharge décorative.
- Pas de jargon technique côté front public.
- Les cartes et listes doivent rester cohérentes entre elles.
