# Product Brief — Annuaire Dog Friendly 974

## Vision
Construire la référence locale fiable pour savoir où aller avec son chien à La Réunion.

## Problem
L'information "chiens acceptés ou non" est fragmentée, souvent non datée, contradictoire et rarement contextualisée.

## Product answer
Un annuaire web mobile-first qui combine :
- fiches éditoriales
- carte interactive
- contributions publiques modérées
- enrichissement data contrôlé
- score de confiance visible

## Guiding principle
On construit un annuaire éditorial augmenté par la data, pas un robot qui devine.

## What the public should experience
- recherche simple
- filtres utiles
- lecture très rapide d'une fiche
- statut clair
- conditions claires
- date de vérification
- confiance visible
- correction simple si erreur

## What the admin team should experience
- file de modération claire
- édition rapide
- fusion de doublons
- suivi des rechecks
- import de candidats exploitable
- audit trail

## MVP public features
- page d'accueil
- page annuaire
- page carte
- fiche lieu
- formulaire "ajouter un lieu"
- formulaire "corriger une fiche"
- pages méthodologie / mentions / contact

## MVP admin features
- dashboard
- liste des fiches
- éditeur de fiche
- gestion des sources
- file des contributions
- file des doublons
- file des revalidations

## Explicit exclusions
- app mobile native
- messagerie
- réservation
- programme de fidélité
- gamification
- avis publics non modérés
- publication automatique par IA

## Public status model
- `allowed`
- `conditional`
- `disallowed`
- `unknown`

## Public trust model
- confiance élevée
- confiance moyenne
- confiance faible

## Editorial tone
Clair, utile, local, crédible.
Pas corporate.
Pas gadget.
Pas bavard.
