# Project Overview — Annuaire Dog Friendly 974

## One-liner
Annuaire web dog-friendly local pour La Réunion, combinant édition humaine, cartographie et enrichissement data sous contrôle.

## Product vision
Le produit doit permettre à un utilisateur de comprendre très vite :
- si un lieu accepte les chiens,
- sous quelles conditions,
- à quel point l'information est fiable,
- quand elle a été vérifiée,
- sur quelle base elle repose.

## Core promise
Trouver rapidement une information crédible sur l'accueil des chiens, sans avoir à croiser 5 plateformes contradictoires.

## Product doctrine
Le projet est un **annuaire éditorial augmenté par la data**.
Les sources externes servent à découvrir, préremplir, scorer et prioriser.
Elles ne doivent jamais publier seules.

## Target audiences
1. habitants de La Réunion avec chien
2. touristes / visiteurs
3. établissements souhaitant être référencés
4. équipe éditoriale / admin

## MVP boundaries
Le MVP doit rester sobre :
- un site web responsive
- une recherche utile
- une carte
- des fiches
- des formulaires
- un admin métier
- un pipeline d'import encadré

Pas d'app native ni de réseau social.

## Success signals
- les utilisateurs trouvent vite les fiches utiles
- les admins peuvent maintenir la base sans friction
- les données contradictoires sont visibles et traitables
- la base reste fraîche dans le temps
