# SEO Strategy — Annuaire Dog Friendly 974

## SEO objective
Être trouvable sur les requêtes locales réellement utiles, sans tomber dans le spam.

## Page types to optimize
- page accueil
- pages catégorie
- pages commune
- fiches lieux
- quelques pages éditoriales utiles

## Technical basics
- SSR/ISR selon le besoin
- titles et meta descriptions uniques
- canonical
- sitemap.xml
- robots.txt
- structured data quand pertinent
- slugs propres

## Content strategy
Privilégier des pages réellement utiles :
- "où manger avec son chien à Saint-Leu"
- "hébergements acceptant les chiens à La Réunion"
- "plages et balades dog-friendly"

## Structured data
- LocalBusiness / Place selon les cas
- utiliser des données structurées cohérentes
- pour l'hébergement, intégrer `petsAllowed` quand le cas est explicite

## On-page rules for place pages
- statut chien visible au-dessus de la ligne de flottaison
- conditions lisibles
- commune lisible
- date de vérification lisible
- source mentionnée

## Anti-spam rules
- pas de pages vides
- pas de duplication massive
- pas de texte généré creux
- pas de faux contenus de remplissage

## Internal linking
- commune -> lieux
- catégorie -> lieux
- fiche -> commune + catégorie
- pages éditoriales -> fiches pertinentes
